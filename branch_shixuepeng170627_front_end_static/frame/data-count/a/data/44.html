鉴别生态板的质量的注意事项

作者： 发布日期：2017-01-07　访问次数：6


　　1、标准的生态板材料是实木压制而成的，中间木材的拼缝衔接十分严密，切面光滑平整，而伪劣生态板材料的内部材料十分粗糙，中间都是缝隙，易翘曲变形，如果板材里使用了不合格的胶，会带来甲醛的危害。 

　　2、产品重量，不同的产品重量不一样，密度不一样，其质量也不一样。但不是说重的质量就一定是好的，需要结合实际的去查看的。

　　3、外观颜色，产品颜色是基本的分辨手段，因为毕竟是木头，其纹理外观肯定是要像木材才行，但是不同类别的产品，所产生的木纹效果也不一样！ 

　　4、很多家具厂机器封边，如果出现结头不美观，有压扁现象，则生态板本身有问题。在选择家具时，注意封边是否平整。

　　5、横截面，横截面的话要有木质的手感才行，没有木质手感的木头那可能配比的话还不到位，但是粗糙程度是根据机器刀片的好坏而产生的，锋利的刀片切出来的产品很平滑，不锋利的刀片切出来的产品就不行了。
