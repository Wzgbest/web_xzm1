胶合板的应用领域决定其结构特征

作者： 发布日期：2017-01-07　访问次数：5


    不得不说，胶合板的应用领域十分广泛，不仅是包装，其在框架结构上也有非常重要的作用。但两种领域中的LVL胶合板产品结构是不同的，因为胶合板的应用领域决定其结构特征。

    众所周知，在胶合板的制备过程中，有结构用与非结构用两种，在实际的生产过程中，我们知道是由单板热压拼合而成的，其不同的应用领域决定其在热压的拼接上表现出不同的结构特征，使之具备相应的使用效果。LVL胶合板在用于结构框架，地板、门窗、建筑桥梁等之类的领域，此应用范围对承受能力、抗压性等方面提出一定要求，相应的为了满足使用之需，在拼接上要求两个相邻的板材在木质纤维上应该保持平衡，各个单板材质应该为同一树种。
