如何处理生态板的特殊污垢

作者： 发布日期：2017-01-07　访问次数：12


    在生态板的日常使用中，要做好日常的维修保养、擦干、上腊等工作，生态板厂表示，在出现各种污染时要及时发现并解决问题。

    对于生态板沾染的无法擦拭的污垢，可以用不易脱毛的棉布沾白电油与水勾兑的混合液清洁，再用棉布沾适量清水清洗，清洗后立刻抹干即可。如果是印泥、口红、墨汁之类可用小量酒精清理，以温和清洁剂、沾湿布拧干后拭抹即可。而铅笔印可用甲醇酒精擦拭。茶、咖啡、强力胶可用甲苯溶剂擦拭；果汁、油漆渍可用丙酮擦拭。

    甲醇、丙酮、甲苯均有不同程度的危险性并对生态板产品的油面处理有腐蚀作用，敬请妥善保管及使用。
