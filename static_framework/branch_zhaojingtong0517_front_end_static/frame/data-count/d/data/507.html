为什么孩子从幼儿园回来总喊“饿”？

时间：2017-02-13

       孩子进入幼儿园以后，总能听到家长向学校反映：“我家宝宝从上幼儿园后，

放学回家就喊“饿”，回到家第一件事就是翻冰箱，找零食。也会有家长说：“幼儿园

每天都有食谱上传到班级群，我也每天都问老师孩子在学校吃饭的情况，老师都说

吃的很好，那这到底是什么原因呢？” 

1
新入园的孩子尤其容易“饿”
 
       首先，孩子刚刚进入幼儿园，在新的环境中，面对新的老师和不熟悉的小朋友，常会感到不安和紧张，有些幼儿的不安和紧张会表现得很强烈。这种紧张的心理状态会在一定程度上抑制幼儿的食欲。　　
 
       其次，有些幼儿在进餐时会表现出心神不定的情况，这也会导致孩子的进食量减少。此时，老师问孩子还要不要添饭时，孩子会说已经吃饱了。当孩子离开幼儿园这个新环境回到家里时，紧张和不安消除，食欲也随之产生，这时孩子感觉到饥饿是正常的。
 
2
幼儿园饮食结构与家中不同
 
       幼儿园的饮食结构和幼儿家中的饮食结构有一定的不同，幼儿需要一段时间来适应这种变化。同时，幼儿在家中经常是成人喂，幼儿被动地吃；而幼儿园就需要自己吃，喂养方式的不同导致幼儿“不愿吃”。　　
 
3
幼儿园活动量大
 
        孩子进入幼儿园后，常会被幼儿园的各种玩具和娱乐设施所吸引，同时，老师也会为孩子们准备许多游戏活动，使孩子们能够更快的适应到幼儿园的环境当中。
 
        当活动量大时，孩子身体的能量消耗也会随之加快。即使，下午幼儿园为孩子准备午点，通过一段时间的活动孩子身体能量消耗后也会使其感到“饿”。
 
4
“大锅饭”不比家中饮食精细
        幼儿园的饭菜总比不家里的精细，如今的家长都比较宠爱孩子，一些孩子偏食、挑食现象严重，这也导致他们不喜欢幼儿园的饭菜，所以会吃得比较少。家长要注意和老师沟通，尽量让孩子养成不偏食、不挑食的好习惯！ 

孩子从幼儿园回来，晚餐该如何安排呢？


        俗话说：“晚餐要吃少。”但这对于正处于生长发育旺盛时期的孩子来说，是不对的。

“因为晚餐距离第二天的早上相隔10小时左右，虽然说睡眠时无须补充食物，但孩子的生

长发育却一刻也不会停止，夜间也是一样，仍需一定的营养物质。”可见，孩子的晚餐不

仅不能少吃，还应吃饱吃好。


        对孩子的晚餐安排，首先要考虑的是热量，即晚餐热量应高些，约占全天热量的40%

为宜，少食高脂类或不易消化的食物，提高晚餐质量可有效地改善人的营养状况。

 

       因此，身体较瘦弱的孩子，家长更应重视孩子的晚餐，利用改善晚餐质量来提高孩子

的体质。相反，若孩子体重已经超重甚至发胖，则应坚持“晚餐吃少”的原则。

 

       一般来说，晚餐热量为全天进食热量的25%—30%为宜，但这个“少”指的是热量要少，

而不是减少数量，因为数量少了，孩子吃不饱，还会影响晚上的睡眠质量。　　　　

 


        如果孩子晚上还喊饿，可以适当在睡前1小时（即夜宵），增加1—2次加餐。

加餐的食物要有营养，比如牛奶、面包、少许的坚果（花生、核桃等）、水果等，避

免给孩子一些太肥腻的食物或那些含盐含糖较多的零食。

 

                             在孩子入园前，家长要培养孩子良好的进餐习惯，

                                                让孩子自己吃，愿意吃；

                                      对于孩子入园出现的饮食上的波动，

                                                  家长不必太着急，

                                     回家后可以适当给孩子再添加些食物，

                                               并和老师进行及时沟通，

                                       了解孩子饮食波动的真正原因，

                                               从而采取正确的对策。
