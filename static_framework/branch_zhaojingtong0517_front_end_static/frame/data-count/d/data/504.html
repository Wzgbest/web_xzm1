时间：2016-12-09

当我们伴着冬日的阳光，快乐的享受生活时，当我们尽情欢笑，快乐的学习和工作时，有一群孩子在等待着社会各界人士的关注，2016年12月9日真爱幼儿园全体教职工及家长、学生举行了一场温暖的爱心捐款活动。

爱心无价，真情永存，虽然这些捐赠，对于福利院的孩子来说是短暂的幸福，但我们相信，大家的爱心，一定会为孩子们送去温暖，愿我们的点滴付出，能改变福利院的孩子们。
