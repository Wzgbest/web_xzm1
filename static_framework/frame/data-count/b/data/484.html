生态板

作者： 发布日期：2016-11-26　访问次数：54
本文网址： http://xinshijibancai.com/info/484.html

      生态板，分狭义和广义两种概念。1.广义上生态板等同于三聚氰胺贴面板，其全称是三聚氰胺浸渍胶膜纸饰面人造板，是将带有不同颜色或纹理的纸放入生态板树脂胶粘剂中浸泡，然后干燥到一定固化程度，将其铺装在刨花板、防潮板、中密度纤维板、胶合板、细木工板或其他硬质纤维板表面，经热压而成的装饰板。2.狭义上的生态板仅指中间所用基材为多层实木(胶合板)、细木工板(如马六甲、杉木、桐木、杨木等)的三聚氰胺饰面板。主要使用在家具、橱柜衣柜、卫浴柜等领域。


公司主要原料及产品

      主要原料：实木板芯、实木皮、生态纸

      主要产品：马六甲生态板、桐木生态板、松木生板板

生产工艺特点

      公司外购进口实木木材、外购国产实木做原料，砂光处理平整——串胶加实木皮——冷修冷压——热压——批灰——砂光找平——串胶贴大木皮——贴生态皮——锯边包装——款到发货

产品细分：

      1、马六甲；所用芯板材料，主要产自马来西亚、印尼、菲律宾等，因途经马六甲海峡，统称马六甲；

      2、桐木、松木、杨木、多层板条、整体多层、5厘背板

      3、按纸的花色分为：枫木、水曲柳、橡木等数十种。
