做家具，就选南山木歌     高端品质，南山木歌板材

木好板才好，南山木歌只做好板材


      “南山木歌”给人第一印象是“山”和“歌”， “南山”会让人联想到陶渊明：采菊东篱下，悠然见南山。又有：福如东海，寿比南山。让人的感觉是自在、洒脱、长久之意。而“木歌”给人一种生命力，活力，绿色，希望。整个名字给人一种积极向上，充满朝气的阳光感。

      在设计标志的时候，更加注意突出这种内在的向上精神。

方案一：文字与图形结合


      该方案突出两大特点：一“山”，二 “歌”的旋律引申而来的曲线。

      通过“南”“木”首字母拼音“n,m”组合，形成山的形状，来照应“南山”，通过在山的表面添加了两条曲线来表达歌的优美旋律。并且也增加了标志整体的立体感。

方案二：中国风

      该方案采用古代印章与毛笔字的结合，突出一种古典韵味。标志背景为山，边缘做成水沁效果，犹如古代印章也如水墨。“南山木歌”采用毛笔字书写，与标志的背景达到了形式上统一的效果。


方案三：文字与字母的结合


                 


     此方案设计更加现代化，把“南山木歌”每个字的拼音首字母拿出来组合成一个标志，从形式上更加现代。为了打破颜色上的单调，在部分字母的处理上做了小调整，增加了橙色，使整体稳重而不古板。
