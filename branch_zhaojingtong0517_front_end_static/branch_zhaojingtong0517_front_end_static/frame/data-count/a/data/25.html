如何辨别实木多层板材的优劣呢？

作者： 发布日期：2017-01-07　访问次数：6


多层板也叫多层实木，学名叫胶合板。

这类板材是由木材旋切或者刨切成薄片，再用胶粘剂胶粘、压制而成的三层或多层的板材，板材之间相邻两块的纤维是相互垂直胶合的，这样做的好处就是增加强度，所以，多层板的受压承重和抗变形能力都大于细木工板或其他实木板材。由于的确用到的也是实木切片，所以冠以实木的名称。

目前这种板材越来越多的运用于定制衣柜和橱柜领域，大有超越刨花板和密度板的趋势。

如何辨别这种板材的优劣呢？

还是眼观、鼻闻、手摸吧。

眼观板材横切面，胶合是否紧实，木材成色，密度是否合格，排列是否美观。鼻闻主要是吻有无刺激性气味。（这点很重要，刺激性过大意味着胶粘不环保，慎选，有鼻炎，鼻子不灵敏的建议先换个鼻子。）手摸主要是摸表层是否平整，无毛刺。

总之，在这个看脸的互联网时代，颜值很重要，连产品做出来都不美观，就更不要说其他了。

多层板更多作为家具等木作产品的基材存在，所以，通常很少单独售卖。现场加工，我们以多层板的衣柜为例，报价通常在300左右-500左右每平方不等，通过价格也能大概判断品质。顺便补充一个说明，我们通常所说的的实木复合地板，所用到的基材就是多层板。

多层实木，从今天起，你就知道它的真面目了。和材料商谈价的时候，如果你很懂，自然会掌握更多的话语权，比如对方告诉你，他这个贵的原因是实木板，多层实木，你就淡淡的说一句，不就胶合板嘛，还这么贵。对方一听，必然肃然起敬。他要和你矫情，把这篇文章给他看，让他也学着点。
