真爱简介

真爱教育集团成立于2003年，是一所集教育培训、教育产品研发和教育服务等于一体的综合性教育机构，真爱教育集团先后打造了真爱亲子园、真爱幼儿园、真爱脑潜能教育三大品牌，形成了独具特色的0-6岁一体化早期教育模式。如今，真爱教育业务涵盖亲子园教育、幼儿园教育、家庭教育、脑潜能开发等多个领域。
真爱教育遍布全国30多个城市，拥有近百家高品质幼儿园， 拥有2千余名教职工，真爱教育专业、完整、高品质的教育服务得到了社会各界的认可，荣获政府、协会、媒体评选的多项荣誉。

发展历程

2003

真爱教育集团投身于幼儿园行业，随后创建了两所直营幼儿园，并且迅速在菏泽市辐射。

2004

创建了菏泽市第一所分园

2005

创建了菏泽市牡丹区第二所分园

2006

创建了菏泽市开发区第三所分园

真爱理念

企业目标：打造中国全脑教育第一品牌
企业使命：为真爱求发展、为家人创价值、为教师谋利益、为社会做贡献
战略思想：真爱教育品牌化、真爱项目综合化、真爱课程特色化、真爱市场国际化、真爱运营产业化、真爱服务增值化
企业精神：做事先做人，做人先感恩
加盟理念：品牌+课程+运营+服务=加盟
营销理念：学生第一、家人第二、真爱第三
团队理念：团队歌（自创）
