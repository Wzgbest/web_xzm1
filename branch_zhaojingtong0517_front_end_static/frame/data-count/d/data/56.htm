真爱教育

创于2003

-

 2003年 真爱教育集团成立，是一所集教育培训、教育产品研发和教育服务等于一体的综合性教育机构。

 2003年 真爱教育集团投身于幼儿园行业，随后创建了两所直营幼儿园，并且迅速在菏泽市辐射。

 2004年 创建了菏泽市第一所分园

 2005年 创建了菏泽市牡丹区第二所分园

 2006年 创建了菏泽市开发区第三所分园

 2007年 创建了菏泽市开发区第四所分园

 2008年 创建了菏泽市曹县第五所分园

 2009年 创建了菏泽市东明第六所分园

 2010年 创建了菏泽市东城办事处第七所分园

 2011年 创建了菏泽市郓城第八所、第九所分园

 2012年 创建了菏泽市牡丹区第十所、第十一所分园

 2013年 真爱教育集团开始做脑潜能开发课程，随后脑潜能开发课程迅速在全国迅速漫延，仅有一个月的时间真爱教育集团就有30余家加盟园。

 2015年 真爱教育集团又创建了牡丹公馆园第十二所分园，仅用两年的时间迅速入驻各大地区，就有100余家加盟园加入真爱教育集团开展脑潜能开发课程。

 2016年 真爱教育集团进入腾飞阶段。
