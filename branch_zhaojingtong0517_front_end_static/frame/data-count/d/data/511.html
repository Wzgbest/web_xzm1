真爱教育【团训篇】

时间：2017-02-17

        幼儿教师是幼儿园直接服务孩子的第一人，承担着孩子命运的责任，承担着

这些孩子家庭幸福的责任，今天我们培养一个孩子成才，能够幸福三代人，我们

培养的孩子成才了，老师的心里会非常高兴，更是每位家长的心愿！

有文化、有精神、有组织、有纪律、有追求、有信仰的一群人才能称之为团队。

一个成功的团队是时时刻刻充满活力的团队，是最具学习力、最具凝聚力的团队！ 

什么是责任！

勇于承担责任！

        团训是一个共赢、共享的活动！真爱的家人们让我们一起加油！！！

        我们通过团训增强了老师的责任心，增强了老师的团队凝聚力，提高了老师

的执行力，转变了老师的思想观念，增进了老师之间的沟通配合，让老师的心灵

得到震撼，让每位老师认识到自己拥有无穷的潜质，增强了老师的自信，让他们

用更强的力量去迎接新的挑战与考验。老师的整体素质提升了，对孩子的照顾更

用心、更体贴，对教学更有激情、更有创意，家长也更加满意。整个幼儿园的发

展更和谐了！
